Laravel API is developed via XAMPP and MYSQL database.

Database name: twelfthman

When project fetched, run:

php artisan migrate
php artisan db:seed
php artisan storage:link

* When seeding, images from cloudinary are downloaded, on every next image upload through application, resized images will be created by code.
